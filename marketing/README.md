# 44Stems — Marketing Hub

Dossier centralisé pour toute la stratégie marketing de 44Stems.
Chaque sous-dossier contient un aspect de la stratégie.

## Structure

```
marketing/
├── competitors/          # Analyse concurrentielle
│   ├── COMPETITOR-INTELLIGENCE-REPORT.md   # Pubs actives des concurrents (Meta, Google, TikTok)
│   └── COMPETITIVE-GAPS.md                 # Opportunités identifiées vs concurrents
├── influencers/          # Stratégie influenceurs & contenu
│   ├── INFLUENCER-CONTENT-STRATEGY.md      # Stratégie complète TikTok/YouTube/UGC
│   └── MICRO-INFLUENCERS.md                # Nano/micro influenceurs abordables (<200 EUR)
├── ads-creative/         # Créas publicitaires & benchmarks
│   └── AD-CREATIVE-PLAYBOOK.md             # Formats, CTR, CPV, best practices
├── community/            # Reddit, Discord, forums, guerrilla marketing
│   └── COMMUNITY-STRATEGY.md               # Subreddits, groupes FB, stratégie "free stems"
└── seo-keywords/         # Mots-clés, SEO, Google Ads
    └── (à venir)
```
